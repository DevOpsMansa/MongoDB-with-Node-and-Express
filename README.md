# MongoDB With Node and Express

## Objectives

* Summarize the benefits of using MongoDB alongside Node and Express.
* Install the MongoDB driver via npm.
* Connect to a MongoDB database in Node.
* Create a database connection module and export the connection.
* Read data from a MongoDB database into Node.
* Insert data into a MongoDB database from Node.
* Update data within a MongoDB database from Node.
* Delete data from a MongoDB database from Node.
* Produce a functional CRUD API using MongoDB and Express.
